# 目标
在任何情况下阻止程序错误和人为误操作将会造成的撞机行为.
面向实际情况, 追求简单高效, 通俗易懂, 便于理解和普及.

# 思路

绑定 运动前预测 碰撞结果拦截器 
运动中实时监测 实时速度计算 碰撞预测 
监测频率 事件的设计
